  <name>Task 1: Fix customer account page (Drizzle to Supabase REST)</name>
  <files>app/[region]/account/page.tsx</files>
  <action>
Replace Drizzle DB queries with Supabase REST queries. The page already uses `getUser()` from `lib/db/queries` for auth (which works — it uses getSupabaseClient internally). The problem is the raw `db` import from `@/lib/db/drizzle` used for profile and order queries.

Replace imports:
```typescript
// REMOVE these:
import { db } from '@/lib/db/drizzle';
import { orders, profiles } from '@/lib/db/schema';
import { eq, desc } from 'drizzle-orm';

// ADD this:
import { getSupabaseClient } from '@/lib/db/supabase';
```

Remove the `if (!db)` guard (no longer needed — getSupabaseClient always returns a client).

Replace the profile query:
```typescript
// Old:
const profileResult = await db.select().from(profiles).where(eq(profiles.userId, user.id)).limit(1);
const profile = profileResult[0] || null;

// New:
const supabase = getSupabaseClient();
const { data: profileData } = await supabase
  .from('profiles')
  .select('*')
  .eq('user_id', user.id)
  .limit(1)
  .single();
const profile = profileData || null;
```

Replace the orders count query:
```typescript
// Old:
const allOrders = await db.select().from(orders).where(eq(orders.userId, user.id));
const totalOrderCount = allOrders.length;

// New:
const { data: allOrders } = await supabase
  .from('orders')
  .select('id')
  .eq('user_id', user.id);
const totalOrderCount = allOrders?.length || 0;
```

Replace the recent orders query:
```typescript
// Old:
const recentOrders = await db.select().from(orders)
  .where(eq(orders.userId, user.id))
  .orderBy(desc(orders.createdAt))
  .limit(3);

// New:
const { data: recentOrders } = await supabase
  .from('orders')
  .select('*')
  .eq('user_id', user.id)
  .order('created_at', { ascending: false })
  .limit(3);
```

Update template references from camelCase to snake_case for Supabase REST response:
- `order.orderNumber` -> `order.order_number`
- `order.total` stays the same
- `order.status` stays the same  
- `order.id` stays the same

For the ReferralCard props, map from snake_case:
- `profile?.retailReferralCode` -> `profile?.retail_referral_code`
- `profile?.wholesaleReferralCode` -> `profile?.wholesale_referral_code`
- `profile?.commissionEarned` -> `profile?.commission_earned`

Ensure `recentOrders` defaults to `[]` if null: `const ordersList = recentOrders || [];` and use `ordersList` in the map.
